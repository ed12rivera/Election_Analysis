Python


LISTS []

list = [item1, item2, item3]
empty list => my_list = [] OR my_list = list()

to get an item from a list => list[index]
			   OR list[index1 : index2]

len - find the length of the list = len(list_name)
append - adds item to end of list = list.append(new_item)
insert - adds item to a list where indicated = list.insert(index, new_item)
remove - removes stated item from the list = list.remove(item_to_be_removed)
pop - removes item at a given index and returns removed item = list.pop(index)

change an element in the the list = list[index] = new_item_desired_at_index


TUPLES ()

tuple = (item1, item2, item3)
empty tuple => my_tuple = () OR my_tuple = tuple()

len - find length of tuple = len(tuple_name)

to get an item from a tuple => tuple[index]
			    OR tuple[index 1 : index 2]


DICTIONARIES {}

dictionary = {key1:value1, key2:value2}
blank_dictionary = {} OR blank_dictionary = dict()

add new item => dictionary["new_key"] = new_value
len - find length of dictionary = len(dictionary_name)
items - returns a list of tuples with key and value = dictionary.items()
keys - returns all the keys in a dictionary = dictionary.keys()
values - returns all the values in a dictionary = dictionary.values()
get - returns value of specified key = dictionary.get("key")

Another method to get value of specified key => dictionary["key"]

list of dictionaries => list_name = [{key1:value1, key2:value2}, {key1:value3, key2:value4}, {key1:value5, key2,value6}]


FOR LOOP

looping through keys in a dictionary: for i in dictionary_name.keys():
					  print(i)

looping through the values in a dictionary: for i in dictionary_name.values():
						print(i)
	OR
					    for value in dictionary_name:
						 print(dictionary_name[value])
	OR
					   for value in dictionary_name:
						 print(dictionary_name(value))

looping through both key and value in a dictionary: for key, value in dictionary_name.items():
							print(key, value)

looping through dictionaries in a list: for dictionary in list_name:
						print(dictionary)

to get just the values of dictionaries in a list: for dictionary in list_name:
							for value in dictionary.values():
								print(value)

to print only specified values from dictionaries in a list: for dictionary in list_name:
								print(dictionary["key_related_to_desired_value])
